Perguntas frequentes

Como funciona?

A Dados Faceis é uma plataforma onde você poderá filtrar e exportar leads de empresas. Você filtra as empresas por ramo de atividade, cidade, estado, porte ou data de abertura. Os dados das empresas filtradas podem ser visualizados em tela ou exportados em formato excel.

Quais dados estão disponíveis?

Nossa plataforma vende dados de empresas (pessoa jurídica), e compreende as seguintes informações: Dados de cadastro da empresa, telefones, emails, endereço e nome dos sócios.

Como faço pra efeturar compras dentro da plataforma?

É bem simples! Basta escolher a quantidade de empresas, efetuar cadastro e realizar o pagamento. O pagamento pode ser feito via PIX, Boleto Bancário ou Cartão de Crédito. As empresas compradas na plataforma não expiram.

De onde vem esses dados?

Os dados filtrados na plataforma B2bleads são dados públicos, disponíveis na Receita Federal, sites de buscas, sites das próprias empresas e dados informados no mercado. Os dados de empresas (pessoa jurídica) dados não são dados sensíveis, nem dados que ferem a Lei Geral de Proteção de Dados (LGPD).
